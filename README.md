# Development

## Setup

### !. Clone/Download this repo

```sh
gh repo clone Tetrax-10/Spicetify-App
```

### 2. Install Dependencies

```sh
npm install
```

<br />

## Development / Build

### 1. Run in development mode

```sh
npm run dev
```

<br />

### 2. Build App

**Windows OS**

```sh
npm run build:win
```

**Mac OS**

```sh
npm run build:mac
```

**Linux**

```sh
npm run build:linux
```

<br />
